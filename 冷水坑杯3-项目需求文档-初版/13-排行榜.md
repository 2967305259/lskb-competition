# 排行榜

## 排行榜体系

系统提供三层排行榜展示：

### 1. 首页 TOP 10（快速预览）
路由：`GET /` → 首页底部

- 显示积分前10名的队伍
- 展示：排名、队伍名、队长、积分
- 点击可跳转至完整排行榜

### 2. 赛事大厅排行榜（实时）
路由：`GET /tournament/` → 赛事大厅

- 完整统计数据
- 所有队伍的实时排名
- 晋级队伍高亮展示
- 最近完成的比赛

### 3. 完整排行榜
路由：`GET /tournament/rankings`

- 所有队伍完整排名
- 显示：排名、队伍名、队长、成员数、队伍总分
- 按 (team_score DESC, special_score DESC, minor_score DESC) 排序

### 4. 最终结果（赛事结束后）
路由：`GET /tournament/results`

- 冠军：第1名队伍
- 亚军：第2名队伍
- 季军：第3名队伍
- 完整排名列表

## 排名计算

### 排序规则
```sql
ORDER BY team_score DESC, special_score DESC, minor_score DESC
```

### 队伍积分构成
```
team_score = sum(所有成员 final_score) × bonus_score(默认500)
```

### 个人积分构成
```
final_score = sum(各场比赛 raw_score × difficulty_multiplier)
```

## 晋级机制

- 管理员设置 `advance_count`（晋级队伍数量）
- 排名前 N 的队伍自动晋级
- 晋级队伍在赛事大厅特殊标记
- 赛事结束后最终排名即晋级结果

## 数据刷新

- 积分实时更新（比赛结果提交时即时计算）
- 排行榜查询即刷新（无缓存）
- 管理员可手动重算全部积分

## 排行榜不足与改进建议

1. **special_score 和 minor_score 未使用**：排名公式包含这两个维度但始终为0，应实现（如：special_score = 全胜奖励，minor_score = 净胜分）或移除
2. **缺少个人排行榜**：只有队伍排名，应增加个人积分排名
3. **缺少肉鸽主题排行榜**：不同肉鸽主题的选手独立排名
4. **历史排名不可查**：赛事结束后无法看到赛中的排名变化过程
5. **排名变化可视化**：缺少排名上升/下降的动态展示
6. **缺少排名筛选**：无法按肉鸽主题、赛程阶段筛选排名
7. **导出功能缺失**：无法导出排行榜数据（CSV/Excel）
