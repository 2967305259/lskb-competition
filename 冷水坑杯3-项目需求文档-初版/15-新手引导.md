# 一、总体目标

整个新手教程采用**页面引导模式**。

不是在当前页面不断跳转，而是：

**每介绍一个功能，就自动跳转到对应页面。**

用户始终知道：

现在在哪里。

这个页面有什么作用。

下一步该做什么。

整个教程结束后，引导用户进入：

**我的（个人中心）**

完成资料填写。

整个过程形成完整闭环。

# 二十二、验收标准

完成后必须满足以下要求：

- 首次登录自动启动教程。
- 教程按照 **首页 → 队伍列表 → 赛事规则 → 赛事大厅 → 排行榜（可选）→ 个人中心 → 填写资料** 的顺序自动切换页面。
- 每一步进入对应页面后，自动滚动到目标区域并显示聚光灯。
- 页面使用真正的挖空遮罩（Spotlight），目标区域完全透明且可点击。
- Tooltip 自动定位，不遮挡目标元素，不超出浏览器边界。
- 支持上一步、下一步、跳过教程、重新观看教程。
- 教程完成后记录状态，不会重复自动播放。
- 所有动画流畅，窗口缩放、滚动后仍能正确定位。
- 不修改、不影响现有业务逻辑，仅作为独立的新手引导模块集成。
