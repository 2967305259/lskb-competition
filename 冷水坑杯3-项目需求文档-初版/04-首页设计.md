# 首页设计

## 首页路由
`GET /` → `main.index` → 公开访问

## 页面组件

### 1. Hero 轮播区
- 自动播放的轮播图组件
- 支持淡入淡出切换动画
- 指示器圆点导航 + 触摸滑动支持
- 自动预加载前后图片
- 支持图片和视频两种媒体类型
- 数据源优先级：CarouselItem 数据库表 > `image/` 目录静态文件
- **当前状态**：仅使用 `image/` 目录下的静态文件（1.png ~ 5.png），CarouselItem 模型已有但无管理界面

### 2. 赛事统计面板
- 参赛队伍数量
- 参赛选手数量
- 已完成的比赛数量
- 当前赛事状态标签

### 3. 最新队伍展示
- 最近创建的队伍列表
- 显示队伍名、队长、创建时间

### 4. TOP 10 排行榜
- 队伍排名前10名
- 显示排名、队伍名、积分

### 5. 导航栏
- 品牌 Logo / 赛事名称
- 首页 / 赛事大厅 / 队伍列表 / 规则
- 登录/注册 或 用户信息下拉菜单
- 通知铃铛图标（带未读数量角标）

### 6. 页脚
- 赛事版权信息
- 相关链接

## 技术实现
- Bootstrap 5 响应式布局
- 自定义暗色主题（明日方舟风格）
- 轮播图：自定义 JS（`carousel.js`）+ CSS（`carousel.css`）
- 模板上下文注入：`unread_notifications_count`

## 首页设计不足与改进建议

1. **CarouselItem 管理界面缺失**：需添加后台轮播图 CRUD 页面
2. **赛事动态信息流**：首页可增加最新比赛结果/动态 feeds
3. **快速入口**：为已登录用户提供快捷操作入口（如：进入我的队伍、查看我的比赛）
4. **SEO 优化**：添加 meta 标签和社交媒体预览信息
5. **移动端优化**：轮播图在移动端的触摸体验可进一步优化
